프로젝트 관리 API

  • POST /projects - 프로젝트 생성
  • GET /projects - 프로젝트 전체 조회
  • GET /projects/{id} - 프로젝트 단일 조회
  • PATCH /projects/{id}/status - 프로젝트 상태 수정
  • PATCH /projects/{id}/content - 역할 배분 내용 수정
  • DELETE /projects/{id} - 프로젝트 삭제

지원서 관리 API

  • POST /projects/{id}/applications - 프로젝트 지원서 작성
  • GET /projects/{id}/applications - 프로젝트 지원서 목록 조회
  • GET /applications/{id} - 지원서 상세 조회
  • PATCH /applications/{id}/status - 지원서 상태 변경
  • DELETE /applications/{id} - 지원서 삭제 (지원 철회)

AI 분석 API

  • POST /analysis/application/{applicationId} - AI 분석 실행
  • GET /analysis/application/{applicationId} - 분석 결과 조회
  • POST /analysis/project/{projectId}/role-assignment - 팀 역할 분배

사용자 관리 API

  • GET /auth/me - 내 프로필 정보 조회
  • PATCH /users/profile - 프로필 정보 수정
  • GET /users/projects - 내가 생성한 프로젝트 목록
  • GET /users/applications - 내 지원서 목록 조회
  • DELETE /auth/logout - 로그아웃
